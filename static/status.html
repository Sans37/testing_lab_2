🚀 Панель мониторинга сервера

Мониторинг инфраструктуры WebLab#4 в реальном времени

Статус Nginx

Веб-сервер работает и принимает подключения на порту 80. Все маршруты правильно настроены.

Статус API

FastAPI v2 работает на порту 8000. Документация Swagger доступна по адресу /api/v2/docs.

База данных

Подключение к PostgreSQL установлено. База данных "nady_bakery" доступна и отвечает на запросы.

Админ-панель

Интерфейс управления базой данных Adminer доступен по адресу /admin. Требуется аутентификация.
